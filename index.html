Works

Voteat（飲食店投票アプリ）

友達や家族との食事で「お店が決まらない」という課題を解決するためのWebアプリ。
飲食店を検索し、候補から投票して多数決でお店を決めることができるサービスです。

主要機能：
・飲食店検索
・投票投稿／投票機能
・投票結果表示・コメント
・LINEログイン

Ruby on Rails PostgreSQL HTML/CSS JavaScript/jQuery

Voteatのリポジトリ： https://github.com/makoto0177/restaurant_vote

トモツク（イベント管理サービス）

イベント作成、チケット販売、売上管理などを備えたWebサービス。
バックエンドにFastAPI、フロントエンドにReactを使用。

主要機能：
・イベント作成／編集／削除
・チケット販売・決済（Stripe連携）
・参加者管理／売上管理

FastAPI React Stripe Docker MySQL

GitHubアカウント: https://github.com/makoto0177
